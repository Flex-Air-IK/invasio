# Invasio v.1.3
## (Язык: RU 🇷🇺)

Приложение для Андроида, помощающее решать тесты при работе с нейросетями<br>

[ССЫЛКА НА .exe](https://github.com/Flex-Air-IK/invasio/releases/download/v1.3/Invasio_setup.exe)
---
## 🚀 Функции

- Захват скриншотов с Android-устройства через ADB
- Выбор области (ROI) с помощью мыши (масштабируемое окно OpenCV)
- Интеллектуальный OCR-конвейер (pytesseract)
- Удаление перекрывающихся фрагментов при прокрутке
- Режим Conditions (объединение нескольких скриншотов)
- Режим Options (поочерёдный захват до 5 вариантов ответа)
- Экспорт структурированного формата тестов в буфер обмена
- Поддержка английского и русского языков
- GUI интерфейс
- GUI бинды для кнопок
- Бинды Хоткеев
- Экран с донатами
- Ссылки на соцсети

## 🙈 Известные ограничения:
- Требуется ручное переключение режимов
- Точность OCR зависит от качества изображения на экране
- Нестабильная работа при распознавании только чисел и специальных символов (проблема на уровне tesseract)

## 💫 Подготовка телефона к работе
### 🚩!Работает только с Андроидом!
1) Идем в настройки телефона
2) Заходим в раздел "О телефоне", ищем строку "Версия ОС"
3) 7 раз кликаем по "Версия ОС", по итогу должны зайти в режим разработчика
4) Переходим обратно в настройки, ищем строку "Для разработчиков", переходим в нее
5) Прокручиваем вниз, то раздела "Отладка"
6) Включаем тумблеры "Отладка по USB", "Установка через USB", "Отладка по USB (настройки безопасности)"
7) Подключаем телефон к компьютеру через USB

# 🔱 Пайплайн использования
1) Подключение телефона, с которого будут проходиться тесты к компьютеру
2) Запуск Invasio
3) Вход в любую нейронку
4) Размещение трех окон: телефона, нейронки, Invasio - в удобном формате на экране компьютера
5) Запуск теста на телефоне
6) Нажатие кнопки/хоткея Invasio (F1 по умолчанию)  для выделения условия теста->выделение->Enter. Выделенный текст отображается
в программе. Если условие длинное, то выделяется часть, затем снова нажимается кнопка/хоткей для условия, и проделывается та же операция
7) Нажатие кнопки/хоткея Invasio (F2 по умолчанию) для выделения варианта ответа->выделение->Enter
8) После успешного формирования условия и вариантов - нажатие кнопки/хоткея Invasio (F3 по умолчанию) для копирования условия с пронумерованными вариантами ответа в буфер обмена
9) Вставление условия с вариантами в нейросеть, ожидание ответа. Выбор нужного ответа в телефоне мышью.
